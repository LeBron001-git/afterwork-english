# 线上部署说明

## 推荐：Vercel

项目已经配置好 `vercel.json` 和部署脚本。

### 首次登录

```bash
pnpm exec vercel login
```

按提示登录 Vercel 账号。

### 生产部署

```bash
pnpm deploy
```

部署成功后，Vercel 会输出一个线上网址。把这个网址发给朋友即可。

### 预览部署

```bash
pnpm deploy:preview
```

## 朋友远程使用

朋友不需要和你在同一个 Wi-Fi。只要打开线上网址即可。

当前版本没有账号系统，学习数据保存在每个访问者自己的浏览器里：

- 你的浏览器保存你的记录
- 朋友 A 的浏览器保存朋友 A 的记录
- 朋友 B 的浏览器保存朋友 B 的记录

## 商业化升级路线

正式商业化前建议分阶段补齐：

1. 用户账号：邮箱、手机号或微信登录
2. 云端数据库：保存学习记录、收藏、错题和复习计划
3. 订阅付费：免费版、会员版、企业版
4. 内容后台：管理单词、语法、课程和场景
5. 数据合规：隐私政策、用户协议、数据删除与导出
6. 域名品牌：绑定正式域名和品牌落地页
7. 运营分析：留存、活跃、学习完成率、转化率
